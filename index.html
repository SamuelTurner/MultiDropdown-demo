MultiDropdown

MultiDropdown is a plugin for inuit.css. Simply place the _multidropdown-core.scss file somewhere in your SASS project, include it somewhere else and extend it where required. In this demo I've followed inuit.css's file structure and extended it within my .page-header.

In this example/demo, I have used MultiDropdown on the header nav above, and extended it as simply as I can. I've used SamuelTurnerUI to build this demo page.

Source

Please view the source code and download from Github.

Support

MultiDropdown works with most modern browsers, and even IE8. It appears to function OK in IE7 with a couple of quirks.
